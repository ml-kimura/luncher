---
id: 3
title: 要件にあったグループを自動編成できる
priority: medium
actor: Batch
integrations:
  - API
  - DB
  - Slack
outputs:
  - batch
  - api
  - db
story:
  who: システム（日次バッチ）
  want: 参加者をコラボランチの要件にあわせてマッチングし、結果をDBに登録してSlackへ通知したい
  why: 日次バッチでマッチング結果のDB登録とSlack通知を漏れなく実施するため
trigger: 毎日11:55（JST）にスケジューラ（cron）から日次バッチが起動される
postConditions:
  - 当日参加者が要件に基づいてグループ編成される
  - 編成結果がDBに保存される
  - 編成結果の通知メッセージがSlackに送信される
acceptanceCriteria:
  - 毎日11:55（JST）のスケジュール時刻に編成バッチが起動される
  - 参加者の数がコラボランチ要件を満たさない場合は編成処理を行わず、処理スキップがログに記録される
  - コラボランチ要件（人数、参加回数）を満たすグループが生成される
  - 編成結果がDBへ保存され、保存失敗時はエラーログが記録される
  - Slack通知がグループごとに送信され、重複通知されない
outOfScope:
  - 参加表明の受付処理
  - 店舗候補の提示および確定
notes:
  - 本USは自動編成の実行と結果通知までを対象とする
