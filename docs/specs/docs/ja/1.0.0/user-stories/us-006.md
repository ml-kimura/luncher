---
id: 6
title: 写真のリプライで経費精算用テキストを受け取れる
priority: medium
actor: Slack
integrations:
  - API
  - DB
  - Slack
outputs:
  - api
  - db
story:
  who: ランチ申請者（支払実行者）
  want: グループ分けSlackメッセージに写真をリプライすることで、親メッセージに写真が表示され、経費精算で入力すべきテキストをSlackリプライで受け取りたい
  why: 経費精算入力を迷わず実施し、入力ミスを減らすため
trigger: グループ分けSlackメッセージに写真がリプライされる
postConditions:
  - 写真付きリプライから、グループ分けSlackメッセージに写真が反映される
  - グループがコラボランチ実施済みステータスにDB上更新される
  - 経費精算用テキストがSlackリプライで送信される
acceptanceCriteria:
  - 写真が添付されたリプライのみ処理対象となる
  - 成功時に親メッセージ上で写真が確認できる
  - 成功時に経費精算用テキストが親メッセージへのリプライとして1回送信される
  - 写真リプライ時に要件を満たしていない時はエラーメッセージを返し、精算テキストを送信しない
outOfScope:
  - 経費申請システムへの自動登録
  - マッチング結果の変更
notes:
  - 本USは精算入力用テキストの生成と通知までを対象とする
