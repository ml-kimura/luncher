---
id: 4
title: グループ割り当て後に欠席できる
priority: medium
actor: Slack
integrations:
  - API
  - DB
  - Slack
outputs:
  - api
  - db
story:
  who: 欠席したいメンバー
  want: 事情により割り当て後に欠席したい
  why: グループ編成後でも欠席を反映し、実態に沿った参加者構成を維持するため
trigger: グループ分けSlackメッセージに欠席リアクションをする
postConditions:
  - 欠席したメンバーが対象グループから除外された状態でDBに保存される
  - 更新後のグループ情報がSlack通知内容に反映される
acceptanceCriteria:
  - グループ分けSlackメッセージへの欠席リアクションで除外処理が開始される
  - 除外対象は割り当て済みメンバーのみとする
  - 除外成功時は更新後グループ構成がSlack上で確認できる
  - 除外後のグループ情報がSlackへ重複なく通知される
  - すでに除外済みのメンバーを再除外しようとした場合は変更せずメッセージを返す
outOfScope:
  - 未割り当てメンバーの新規追加
  - 自動編成アルゴリズムの変更
notes:
  - 本USは欠席によるメンバー除外と結果反映までを対象とする
