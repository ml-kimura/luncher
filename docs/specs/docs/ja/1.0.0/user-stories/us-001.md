---
id: 1
title: 日次のコラボランチ案内をSlackに配信できる
priority: medium
actor: Batch
integrations:
  - API
  - DB
  - Slack
outputs:
  - batch
  - api
  - db
story:
  who: システム（日次バッチ）
  want: 日次でコラボランチ募集メッセージをSlackに配信したい
  why: 当日の参加表明の起点となる案内を定時に提供するため
trigger: 毎日10:00（JST）にスケジューラ（cron）から日次バッチが起動される
postConditions:
  - 当日のコラボランチレコードがDBに登録される
  - 当日のコラボランチ募集メッセージが指定のSlackチャンネルに1回投稿されている
  - バッチ実行結果（成功/失敗）がログに記録されている
acceptanceCriteria:
  - 毎日10:00（JST）のスケジュール時刻にバッチが起動される
  - 指定Slackチャンネルに当日募集メッセージが1回だけ投稿される
  - バッチ実行結果（成功/失敗）がログに記録される
  - 同日内での重複実行時は、同一内容の募集メッセージが重複投稿されない
outOfScope:
  - 日次案内の配信開始以降の処理
notes:
  - 配信時刻変更は運用設定で管理する
---
