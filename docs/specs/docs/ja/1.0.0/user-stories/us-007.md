title: システム管理者が要件設定を変更できる
priority: medium
actor: Web
integrations:
  - API
  - DB
outputs:
  - screen
  - api
  - db
story:
  who: システム管理者
  want: コラボランチの要件にあわせて、設定を変更したい
  why: コラボランチの要件の変更に応じてシステム制約を調整するため
trigger: システム管理者が要件設定画面で保存操作を実行する
postConditions:
  - 更新した要件設定がDBに保存される
  - 次回以降のグループ編成処理に更新後設定が適用される
acceptanceCriteria:
  - システム管理者は要件設定画面で対象設定を編集できる
  - 保存操作時に入力値バリデーションが実行される
  - バリデーションを満たした場合のみ保存される
  - 保存成功後に更新内容が画面上で確認できる
  - 保存失敗時は既存設定を保持したままエラーメッセージを表示する
outOfScope:
  - すでに確定済みの当日グループへの遡及適用
  - 参加表明の受付処理
notes:
  - 本USは要件設定の変更と保存までを対象とする
