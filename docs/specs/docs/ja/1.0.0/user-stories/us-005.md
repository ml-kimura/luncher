---
id: 5
title: 割り当てられたグループに他のメンバーが参加できる
priority: medium
actor: Slack
integrations:
  - API
  - DB
  - Slack
outputs:
  - api
  - db
story:
  who: グループに割り当てられていないメンバー
  want: 自動編成後のグループに対して、後から参加を表明したい
  why: 急な欠席による欠員が発生したグループや参加表明してなかったメンバーが後から参加できるようにするため
trigger: グループ分けSlackメッセージに参加リアクションをする
postConditions:
  - 参加リアクションしたメンバーが対象グループに追加された状態でDBに保存される
  - 追加後のグループ情報がSlack通知内容に反映される
acceptanceCriteria:
  - グループ分けSlackメッセージへの参加リアクションで追加処理が開始される
  - 追加対象は未割り当てメンバーのみとする
  - 追加成功時は更新後グループ構成がSlack上で確認できる
  - 同一メンバーの重複追加は発生しない
outOfScope:
  - 既存メンバーの除外処理
  - 自動編成アルゴリズムの変更
notes:
  - 本USは後参加メンバーの追加反映までを対象とする
