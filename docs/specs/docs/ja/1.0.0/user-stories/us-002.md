title: Slackから参加表明できる
priority: medium
actor: Slack
integrations:
  - API
  - DB
outputs:
  - api
  - db
story:
  who: 当日参加者
  want: Slack からコラボランチへの参加表明をしたい
  why: 参加意思を簡単に共有し、マッチング対象として扱ってもらうため
trigger: Slackのコラボランチ募集メッセージにリアクションする
postConditions:
  - 参加表明イベントがAPIに送信される
  - リアクションしたユーザーが当日マッチング対象としてDBに登録される
acceptanceCriteria:
  - 参加表明操作でAPI呼び出しが実行される
  - 成功時に参加表明完了がSlack上で確認できる
  - 重複表明時は重複登録されない
  - リアクションを解除した場合、対象外になる
outOfScope:
  - グループ編成の実行
  - マッチング結果通知
notes:
  - 本USは参加表明受付までを対象とする
