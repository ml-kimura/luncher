# コスト管理

## 目的と範囲

本ページは **Azure 上のコストをどう管理するか**（可視化・予算・レビュー・是正の運用）のみを扱う。  
SKU 選定や冗長性など **リソース設計そのもの**は、[ネットワーク](./network.md)・[ストレージ](./storage.md)・[監視](./monitor.md) 等各章に記載する。

## ツールと方針

1. **Microsoft Cost Management**：コスト分析、コストアラート、予算（Budget）を利用する。請求単位は、リソースグループ単位でフィルタし、環境ごとに比較。
1. **予算（Budget）**：環境（`prod` / `stg` / `dev`）単位で設定し、**しきい値（例: 50% / 80% / 100%）** で、**Teams に通知**。
1. **Azure Advisor**：コストに関する推奨を **月次**で確認し、採用可否と期限を記録。
1. **タグ**：コスト按分のため、Terraform で付与。タグキー・値は、[タグ](#タグ) 参照。

## 運用サイクル

| 頻度           | 内容                                                                                                           |
| -------------- | -------------------------------------------------------------------------------------------------------------- |
| **月次**       | 前月のコストを環境・サービス別に確認。前月比・異常増加があれば原因を特定）。Cost Management のアラートと突合。 |
| **四半期**     | 未使用・過剰プロビジョニングの棚卸し、Advisor 推奨の消化、予算の見直し。                                       |
| **予算超過時** | アラート発火後、影響範囲と是正案を起案し、承認の上、是正策を実施。緊急時のみ例外を許可し、対処後に報告。       |

## タグ

| タグ名        | 目的                                 | 必須         | 値         | 意味                                                  |
| ------------- | ------------------------------------ | ------------ | ---------- | ----------------------------------------------------- |
| `environment` | 環境分離・コスト按分                 | 必須         | `prod`     | 本番                                                  |
|               |                                      |              | `stg`      | ステージング                                          |
|               |                                      |              | `dev`      | 開発                                                  |
| `app`         | 論理コンポーネントまたは共有系の按分 | 条件付き必須 | `web`      | Web アプリケーション                                  |
|               |                                      |              | `api`      | API                                                   |
|               |                                      |              | `batch`    | バッチ                                                |
|               |                                      |              | `shared`   | 共有リソース                                          |
|               |                                      |              | `platform` | プラットフォーム （システムに直接関係しないリソース） |
